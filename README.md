# Site Blocker

## SOMETHING SIMPLE BTW

Site Blocker is aN extension that allows you to block access to specific websites by setting a password.

## Usage

### 1. Set a Password

-   Click on the **Site Blocker icon** (usually a small shield or custom icon) in your Chrome toolbar.
-   In the popup, enter a password in the "Set password" input field.
-   Click the **"Set Password"** button. This password will be required to access blocked sites.

### 2. Block a Site

-   Navigate to the website you wish to block.
-   Click the **Site Blocker icon** in your Chrome toolbar.
-   Toggle the switch labeled **"Block current site"** to activate blocking for that specific URL.

### 3. Access a Blocked Site

-   When you try to visit a site that is on your blocked list, you will be redirected to a "Site Blocked" page.
-   To gain temporary access, enter your set password in the input field on the blocked page.
-   Click the **"Submit"** button. If the password is correct, you will be redirected to the site.

### 4. Manage Blocked Sites

-   Click the **Site Blocker icon** to open the extension popup.
-   Under the "Blocked Sites" section, you will see a list of all currently blocked websites.
-   To remove a site from the blocked list, click the **"Remove"** button next to its entry.
